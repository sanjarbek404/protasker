# ProTasker - Zamonaviy To-Do List

**ProTasker** bu murakkab bo'lmagan, lekin o'zida katta imkoniyatlarni mujassam etgan zamonaviy to-do list veb ilovasi. 
Bu loyiha foydalanuvchilarning kundalik vazifalarini, loyihalarini samarali boshqarish, hamda ish unumdorligini yanada oshirishiga ko'maklashadi. 
Loyiha ilg'or "Editorial Aesthetic" dizayn andozalari asosida qurilgan!

## 🚀 Asosiy Imkoniyatlar
- **Editorial Dizayn:** Chiroyli qora-oq klassik interfeys (`Playfair Display`, `Inter` shriftlari bilan).
- **Vazifalar boshqaruvi:** Yangi vazifalarni qo'shish, muddat belgilash (kalendar), ustuvorliklarini (past, o'rta, yuqori) sozlash va toifalarga (shaxsiy, ish, loyiha) ajratish.
- **Kalendar funksiyasi:** Vazifalar qaysi kunga rejalashtirilganligini to'liq kalendar ko'rinishida nazorat qilish imkoniyati.
- **Animatsiyalar:** Har bir harakat (`Framer Motion` yordamida) yumshoq va professional animatsiyalar bilan boyitilgan.
- **Profilni sozlash:** Ilova ichida o'z ism-familiyangiz, rolingizni belgilashingiz, hamda rasm yuklashingiz mumkin.
- **Local Storage bilan integratsiya:** Ma'lumotlaringiz ob-havoga qarab yo'qolib qolmaydi :) Barcha ma'lumotlar o'zingizning brauzeringiz xotirasida mukammal shaklda saqlanadi.

## 🛠 Texnologiyalar
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Stilizatsiya:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animatsiyalar:** [Motion (Framer Motion)](https://motion.dev/)
- **Utility / Icons:** `date-fns`, `lucide-react`, `shadcn/ui` komponentlari.
- **Til:** TypeScript

## ⚙️ O'rnatish

Loyihani mahalliy kompyuteringizda ishga tushirish uchun:

1. Repozitoriyni yuklab oling:
```bash
git clone https://github.com/sanjarbek404/protasker.git
cd protasker
```

2. Kerakli paketlarni o'rnating:
```bash
npm install
```

3. Dasturni ishga tushiring:
```bash
npm run dev
```

Keyin, brauzeringizda `http://localhost:3000` manzilini oching va zavqlaning!
